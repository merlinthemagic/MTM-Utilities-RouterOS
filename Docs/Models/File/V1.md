#initialize

```
/import file-name=flash/storage/MTM/Utilities/Enable.rsc;
:global MtmUtils;
:local fileObj [($MtmUtils->"get") "getFiles()->getFile()"];
```

## Methods:

### setPathAsString:

```
:local path "flash/dir1/myFile.txt"; ##path and name of the file
:local bool [($fileObj->"setPathAsString") $path];

```

### getPathAsString:

```
:local str [($fileObj->"getPathAsString")]; ##string "flash/dir1/myFile.txt"

```

### readChunk

```
:local bytes 8; ##how many bytes do we want to read
:local increOffset true; ##default false, should we increment the offset by the byte count after reading?
:local str [($fileObj->"bytes") $bytes ($increOffset)]; ##bytes from the file starting at the current offset, with length of bytes given

```

### terminate

Removes the file object from memory. Important to do after being done or memory will keep being used

```
:local bool [($fileObj->"terminate")]; ##true

```



### getGuid

```
:local str [($fileObj->"getGuid")]; ##string "2B0F4495-01E0-02F7-3E63-6F3436E9C811"

```

### getVersion

```
:local num [($fileObj->"getVersion")]; ##num 1

```

### getInitTime

Time the object was created.

```
:local str [($fileObj->"getInitTime")]; ##str 1778240015.000001

```

### setOffset

```
:local offset 12; ##how many bytes into the file do we want to start reading
:local bool [($fileObj->"setOffset") $offset]; ##true

```
### getOffset

```
:local num [($fileObj->"getOffset")]; ##num current offset we start reading from in the file

```
### addOffset

```
:local increOffset 50; ##how many bytes do we want to add to the current offset
:local bool [($fileObj->"addOffset") $increOffset]; ##true

```
